## project description:
- this is a personal portfolio website built using HTML, CSS, and JavaScript. 

## Features
- About Me section
- Projects section (2 sample projects)
- Contact form (no backend)
- Responsive design for mobile, tablet, and desktop
- JavaScript greeting based on time of day

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git & GitHub

## How to Run Locally
1. Download or clone the repository.
2. open the 'index.html' folder in a browser
